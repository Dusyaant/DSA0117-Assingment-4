This repository contains a comprehensive collection of C++ programs developed for the **DSA0117** class assignment. The projects cover fundamental programming logic, mathematical algorithms, object-oriented programming (OOP), and memory management using pointers.

## 🚀 Repository Highlights

### 1. Fundamental Logic & Loops
*   **Square & Cube Roots:** Implementing root calculations using `if-else` logic and `cmath` while handling validation for negative inputs.
*   **Perfect Number Analysis:** Comparative implementations using `FOR`, `WHILE`, and `DO-WHILE` loops to identify numbers equal to the sum of their proper divisors.
*   **Fibonacci Sequence:** Efficiently finding the $N^{th}$ Fibonacci number using an `IF-ELSE` ladder and iterative logic.
*   **Natural Number Summation:** Accumulating sums for $n$ numbers across different loop structures.

### 2. Array & Bit Manipulation
*   **Smallest Missing Element:** Optimized `WHILE` loop algorithm to detect gaps in sorted numerical sequences.
*   **Most Frequent Element:** Logic-based search to identify the mode within a dataset using frequency counting.
*   **Bitwise Operations:** A technical demonstration of setting the $N^{th}$ bit of an integer using the `|` and `<<` operators.

### 3. Memory & Object-Oriented Programming (OOP)
*   **Pointer Modification:** Demonstrates "Pass by Address" to modify local variables of `main()` from a sub-function (`modifyValues`).
*   **Pointer Arithmetic:** Visualizing how the compiler translates `arr[i]` into `*(ptr + i)` by printing memory addresses and hexadecimal offsets.
*   **Multiple Inheritance:** A class hierarchy (`Person` + `Employee` → `DisplayInfo`) showing data aggregation from multiple parent classes.
*   **Access Specifiers:** A rigorous test of `public`, `private`, and `protected` visibility across base and derived class boundaries.
